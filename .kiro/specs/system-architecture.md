# System Architecture

## Frontend Stack
- React 18
- React Router v6
- Tailwind CSS
- Zustand (state management)
- Vite (build tool)

## Architecture Principles
- Component-based UI
- Domain-separated state stores
- Mobile-first responsive design
- Trust indicators embedded across UI

## High-Level Structure
- Pages: Buyer flows (Home, Product, Checkout, Orders)
- Components: Reusable UI and domain components
- Stores: Product, Order, Escrow, Auth
